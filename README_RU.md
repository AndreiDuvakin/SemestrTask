# Две лисы и двадцать кур

## О проекте

Это графическая настольная игра, реализованная на Python с использованием фреймворка PyQt6.  
Проект выполнен в рамках учебного предмета МДК.02.01 Технология разработки программного обеспечения (специальность 09.02.07 Информационные системы и программирование).

## Правила игры

### Поле
- Игровое поле в виде креста размером 7×7 клеток.
- В начале игры на поле находятся:
  - 2 лисы в верхней центральной части
  - 20 кур, которые распределены по полю

### Ходы
#### Куры (игрок)
- Перемещаются на один шаг:
  - вверх
  - влево
  - вправо
- Не могут двигаться вниз или по диагонали.

#### Лисы (компьютер)
- Перемещаются на один шаг в любом направлении (вверх, вниз, влево, вправо).
- Могут есть кур:  
  Если за курицей (по горизонтали или вертикали) на одну клетку есть свободное поле, лиса перепрыгивает через курицу и съедает её.
- Лиса обязана есть, если есть возможность выбора.

### Победа / Поражение
- Игрок (куры) выигрывает, если куры занимают 9 полей, образующих верхний квадрат поля.
- Лисы выигрывают, если съедят 12 кур (оставшихся кур недостаточно для победы).

## Интерфейс

- Графическое окно с фиксированным размером.
- Кнопки клеток отображают иконки:
  - курица
  - лиса
  - пустая клетка
- При выборе куры подсвечиваются доступные ходы - зелёным.
- После каждого хода игрока автоматически ходят лисы.

## Логика лис

Лиса последовательно пытается:
1. Съесть курицу по всем 4 направлениям прыжка.
2. Если есть невозможно — случайное перемещение на соседнюю свободную клетку.
